# Instrucciones para crear la base de datos Hotel Costa Azul

## Requisitos

Cada integrante debe tener instalado:

- MySQL Server
- MySQL Workbench
- Visual Studio Code
- Python

1. Abrir MySQL Workbench.
2. Entrar a la conexión local de MySQL.
